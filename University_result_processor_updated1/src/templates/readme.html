    • Frequently Asked Questions

    What file formats are supported?

    Currently, we only support PDF files generated from the official MSBTE result portal.

    How accurate is the analysis?

    While we strive for high accuracy, it's recommended to verify critical data with original documents.

    Is my data secure?

    Yes, all processing is done locally and files are deleted immediately after analysis.

    Please report any discrepancies through our contact form with details of the issue.
